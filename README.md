# Whack-A-Bongo (ft. Bongo Cat)
A basic Whack-A-Mole game, but instead of moles, you whack the bongos to help with Bongo Cat's performance!

Deployed at https://achan.dev/whack-a-bongo

Created as part of MintBean Hackathon (6-day timeframe)

## Technology used
* Javascript
* Three.js
* Blender
* Webpack
* Babel
